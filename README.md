# 🤖 Gemini Chat Summarizer Bot (v2.5)

ТГ-Бот представляет собой интеллектуальный инструмент для групповых чатов Telegram, который агрегирует историю переписки и использует нейросеть Gemini 3.5 Flash для её гибкого анализа. Бот умеет адаптироваться под запросы пользователей: от сухой деловой выжимки до агрессивного стендапа или написания кастомных куплетов по мотивам чата.

---

### **✨ Что нового в v2.5**
*   **Тайм-фреймы:** Бот умеет вытягивать сообщения за конкретный прошедший промежуток времени (минуты, часы, дни), динамически вычисляя таймстампы.
*   **Гибкий выбор объема:** Можно запросить анализ по фиксированному числу сообщений (например, последние 150 реплик).
*   **Идентификация по User ID:** Бот отслеживает авторов не по текстовым никнеймам (которые пользователи постоянно меняют), а по уникальному числовому ID Telegram. Это гарантирует 100% точность статистики.
*   **Таргетированный анализ:** Возможность отфильтровать переписку и сделать саммари сообщений только одного конкретного участника через @username.
*   **Генератор ИИ-куплетов (Режим Музыки):** Бот умеет на лету дописывать к результату анализа уникальное стихотворение или панк-куплет, резюмирующий драму в чате.
*   **Лидерборд:** Автоматическое определение самого активного "болтуна" в выборке.
*   **Изоляция чатов:** История сообщений хранится и обрабатывается отдельно для каждого группового чата. Бот никогда не смешает контекст разных групп.

### **📊 Команды и использование**

Команда `/summary` стала максимально гибкой. Аргументы можно комбинировать в любом порядке.

#### 🎭 Режимы работы (Стили)
**Стандартный (normal):** Строгая, лаконичная выжимка. Только факты, суть споров и принятые решения. Имена авторов опускаются для экономии места.
**Ироничный фельетон (j):** Едкая «прожарка» чата. Бот объединяет людей в смысловые группы, высмеивает контекст обсуждения и использует реальные имена и фамилии участников (без скучного цитирования формата «А сказал Б»).

#### 📑 Конструктор команд (Шпаргалка по аргументам)
Аргументы команды /summary можно комбинировать в любом порядке и в любых сочетаниях. Бот сам поймет, что от него хотят.
| Команда | Что произойдет |
| :--- | :--- |
| `/summary` | Краткая суть последних 20 сообщений. |
| `/summary 100` | Анализ последних 100 сообщений. |
| `/summary 1h` | Саммари обсуждения за последний час (доступны `m`, `h`, `d`). |
| `/summary j 30m` | Ироничный пересказ событий за последние 30 минут с упоминанием имен. |
| `/summary s` | Добавляет в конец сообщения ИИ-куплет (красивый стих для normal или дерзкий рэп/рок для j). |
| `/summary @username` | Выборка только по конкретному пользователю. |

#### 💡 Примеры мощных комбинаций в чате:
/summary j s 1h — Сделать смешной пересказ бурных обсуждений за последний час и припечатать всех панк-роком с упоминанием имен.
/summary @s84_wp 5h — Посмотреть, чем занимался конкретный пользователь последние 5 часов в чате и выдать краткую суть его тезисов.
/summary s 300 — Собрать сухую выжимку по крупной ветке обсуждения (300 сообщений) и красиво подытожить её финальным четверостишием.

### **🛠 Установка и развертывание**

#### **1. Требования**
* **Python 3.9+**
* **Gemini API Key** (взять в [Google AI Studio](https://aistudio.google.com/))
* **Telegram Bot Token** (взять у [@BotFather](https://t.me/botfather))

---

#### **2. Быстрый старт на VPS**

# Клонируем и настраиваем окружение
```bash
mkdir smart_bot && cd smart_bot
python3 -m venv venv
source venv/bin/activate
```
# Устанавливаем зависимости
```bash
pip install python-telegram-bot google-generativeai python-dotenv
```
---

#### **3. Настройка переменных**

Создайте файл .env в корне проекта:
```ini
TELEGRAM_TOKEN=ваш_токен_от_botfather
GEMINI_API_KEY=ваш_ключ_от_google
```
___

#### ⚙️ Настройка Systemd (Автозапуск 24/7)

Создайте файл службы:
```bash
sudo nano /etc/systemd/system/smart_bot.service
```
```ini
[Unit]
Description=Gemini Chat Summarizer Bot
After=network.target

[Service]
WorkingDirectory=/home/USER/smart_bot
ExecStart=/home/USER/smart_bot/venv/bin/python3 /home/USER/smart_bot/bot.py
Restart=always
RestartSec=5
User=USER

[Install]
WantedBy=multi-user.target
```
Замените USER на имя вашего пользователя в системе.

Запустите службу:
```bash
sudo systemctl daemon-reload
sudo systemctl enable smart_bot
sudo systemctl start smart_bot
```

### 💾 Технические детали

#### База данных: JSON-файл chat_history.json (хранит последние 1000 сообщений для каждого чата).
#### Защита квоты API: Даже если за указанный промежуток времени (например, 1d) чат написал 2000 сообщений, внутренний предохранитель if len(filtered_messages) >= 300: break обрежет выборку на 300. Это экономит токены и предотвращает падение бота по лимиту символов в Telegram (4000 знаков).
#### Отказоустойчивость Markdown: Если Gemini сгенерирует кривые теги разметки (из-за чего Telegram обычно отказывается отправлять сообщение), бот поймает ошибку, автоматически очистит текст от спецсимволов и отправит его plain-текстом. Самоликвидация исключена.
#### Privacy: Для корректной работы в группах отключите Privacy Mode в настройках @BotFather и дайте боту права администратора.
#### Безопасность: Файл истории и .env должны быть добавлены в .gitignore.
